# matrizagua

Dashboard of Water Flow in Lisbon. Drinking Water, Waste Water and Water Balance.

## SETUP

```
git clone https://github.com/FranMacedo/matrizagua.git
cd matrizagua
python -m venv venv
.\venv\scripts\activate
pip install -r requirements.txt
```

## RUN

```
	python app.py
```
